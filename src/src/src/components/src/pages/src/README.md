# 🤖 AI Career Buddy

AI Career Buddy is an AI-powered career guidance platform built with React, TypeScript, Vite, and Tailwind CSS. It helps students and job seekers build resumes, prepare for interviews, discover jobs, and receive AI-based career guidance.

---

## 🚀 Features

- 🤖 AI Career Chat Assistant
- 📄 ATS Resume Builder
- 📊 Resume Score Analyzer
- 🎤 AI Mock Interview
- 💼 Job & Internship Search
- 🛣️ Career Roadmaps
- 📚 Learning Resources
- 👤 User Profile
- 🔐 Login & Signup
- 🌙 Dark Mode
- 📱 Fully Responsive Design

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ai-career-buddy.git
```

Go to the project:

```bash
cd ai-career-buddy
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root.

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
VITE_API_URL=https://your-api-url.com
```

Access in code:

```ts
const apiKey = import.meta.env.VITE_GEMINI_API_KEY;
```
